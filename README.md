# bootcamp-Hello-World-Python
 Hello World-assignments
 
This repository contains the solutions for the introductory Python programming assignment. The objective of this assignment is to practice fundamental Python concepts including basic output, string manipulation, variable handling, debugging type errors, and string formatting methods.

## Tasks Overview

The script covers the following exercises:

1. **Basic Output:** Printing a literal string to the console.
2. **String Concatenation with Variables:** Displaying strings combined with variables using both comma separation and the `+` operator.
3. **Data Type Constraints & Errors:** Demonstrating how Python handles combining different types (Strings and Integers) and how to fix type mismatches using explicit type casting.
4. **String Formatting:** Using advanced formatting techniques including `.format()` and `f-strings` to inject variables into text templates.
